# PlaceHub Frontend Documentation

Welcome to the PlaceHub documentation. This directory contains guides for developers to maintain and extend the frontend application.

## Table of Contents

1. [**Theme Documentation**](./THEME_DOCS.md) - Details about colors, fonts, and design tokens.
2. [**UI Documentation**](./UI_DOCS.md) - Information about component structure, patterns, and best practices.

## Developer Quick Start

To follow the existing design system:
- Use Tailwind classes with our custom theme: `primary`, `secondary`, `accent`, `background`.
- Use `lucide-react` for icons.
- Use `MainLayout` for new pages.
